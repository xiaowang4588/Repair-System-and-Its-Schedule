## 学生端重构详情

- `pages/course/schedule.vue` — 课表查询+空教室（合并）
- `pages/profile/profile.vue` — 个人中心

### 重构文件
- `pages/index/index.vue` — 首页（用户信息+统计+快捷入口+校园动态）
- `pages/repair/list.vue` — 报修记录（全部/我的切换+状态筛选+详情弹窗+编辑）
- `pages/repair/repair.vue` — 提交报修（优化跳转逻辑）

### 页面结构
| Tab | 页面 | 功能 |
|-----|------|------|
| 首页 | index.vue | 用户信息+报修统计+快捷入口+校园动态 |
| 报修 | list.vue | 提交报修入口+记录查看（全部/我的）+详情+编辑 |
| 课表 | schedule.vue | 课表查询+空教室查询 |
| 我的 | profile.vue | 个人信息+改密+退出登录 |

### API
- GET /api/student/info — 学生信息+统计
- POST /api/student/login — 学生登录
- POST /api/student/change-password — 修改密码
- GET /api/repair/my-list — 我的报修记录
- POST /api/repair/student-update — 学生编辑报修
